== Description ==

Let you create images gallerys and insert them in your post

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug in, you will see an new option called "WP Image Slider".

== Adding a gallery ==
5. Go to the "WP Image Slider" option in the Settings menu.
6. Click in the "Add New Gallery".
   *** Important: You must had been already added images to your word press from post or 
       media menu.   
7. Select the image you want to include in your gallery, set a title, dimentions, and click
   save.

== Using the gallery in your post ==
   
8. Once you have added the gallery, copy and paste the sintax that appear in the screen, 
   that like [getmygallery id=(1)].
    
9. Paste the text in your post text. Done !
